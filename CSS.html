Vademecum o kolorach w CSS

  1. color: brown; /*blisko 200 nazw*/
  1. color: rgb(100,200,45); /*ponad 16 mln kolorów*/
  1. color: #44sh1a; /*system szesnastkowy - ponad 16 mln kolorów*/
  1. color: rgba(20%,67%,16%,90%) /*a- kanał alfa - przeźroczystość, 1 (100%)-nieprzeźroczysty, 0 - pełna przeźroczystość, kolor niewidoczny*/
  1. color: #4e1; /*zapis skrócony #44ee11*/

Monitor ma 3 kolory pixeli - red, green, blue (rgb), które świecą mocniej lub słabiej. Jeśli wszystkie 3 świecą maksymalnie powstaje kolor biały. Czyli monitor zużywa najwięcej prądu gdy jest wyświetlane najwięcej przestrzeni białej, a najmniej przy ciemnych kolorach. Kolor czarny to nieświecący się pixel.

Kolor czarny - żaden piksel nie świeci

color: black;

color: rgb(0,0,0);

color: rgb(0%,0%,0%);

color: #000000;

color: #000;

Kolor biały - wszystkie pixele świecą na maksa

color: white;

color: rgb(255,255,255);

color: rgb(100%,100%,100%);

color: #ffffff;

color: #fff;

Kolor czerwony - pixel czerwony świeci na maksa, pozostałe nie świecą

color: red;

color: rgb(255,0,0);

color: rgb(100%,0,0);

color: #f00;

color: #ff0000;

Kolor szary - pixele świecą z tą samą jednakową mocą, szarość uzyskujemy dając trzy takie same wartości każdemu pikselowi. Im większe wartości tym szarość będzie ciemniejsza, a im mniejsze tym szarość będzie jaśniejsza.

color: red;

color: rgb(80,80,80);

color: #f505050;
