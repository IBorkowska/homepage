JavaScript - wstęp

JS jest językiem programowania, ma więc tak samo jak HTML i CSS swoją składnię, chociaż jest ona nieco bardziej rozbudowana. Odpowiada za interakcje na stronie www.

Kod JS w html-u dodajemy w znaczniku <script>...</script> przed znacznikiem zamykającym </body> lub w osobnym pliku o nazwie main.js

JS jako język daję nam masę narzędzi. Jednym z nich jest polecenie: console.log('Siema!');

Konsola jest miejscem gdzie programiści mogą podejrzeć rzeczy, które nie wyświetlają się bezpośrednio na stronach www, ale mogą im pomóc zrozumieć dlaczego kod nie działa. Konsola znajduje się w DevTools-ach

Podsumowanie

  • Kod JS uruchamia się raz (po załadowaniu strony)
  • Java to nie JavaScript
  • JavaScript to frontendowe programowanie
  • Za pomocą JS można robić różne szalone rzeczy
